# Лабораторная работа 2. Работа с коллекциями

**Тема**: работа с коллекциями

## Цель

Проанализировать и сравнить производительность операций над коллекциями: List, LinkedList, Queue, Stack и ImmutableList.

## Требуемые операции

- Добавление элементов (в конец, в начало, в середину)
- Удаление элементов (из начала, из конца, из середины)
- Поиск элемента по значению
- Получение элемента по индексу (если применимо)

## Объем данных

Работать с коллекциями размером не менее 100 000 элементов для получения релевантных замеров производительности.

## Требования к тестам

- Написать автоматизированные тесты, которые проверяют корректность всех операций (добавление, удаление, поиск) для каждой коллекции.
- Тесты должны быть детерминированными и воспроизводимыми (одинаковые входные данные, одинаковые ожидаемые результаты).
- Использовать фреймворки для тестирования (xUnit).

## Замеры производительности

- Использовать Stopwatch для измерения времени выполнения каждой операции.
- Выполнить каждую операцию не менее 5 раз и усреднить результаты.
- Сравнить время выполнения операций для разных коллекций, сделать выводы.

## Структура проекта

- **src/Lab02** – консольное приложение с основной логикой:
  - `Services/CollectionBenchmark.cs` – класс для проведения бенчмарков.
  - `Services/ReportGenerator.cs` – класс для генерации отчета.
  - `Models/BenchmarkResult.cs` – модели для хранения результатов.
  - `Program.cs` – точка входа приложения.
- **tests/Lab02.Tests** – проект с юнит-тестами (xUnit):
  - `CollectionTests.cs` – тесты для всех коллекций.

## Используемые технологии и библиотеки

- **.NET 8.0**
- **System.Collections.Immutable** для ImmutableList<T>
- **xUnit** для юнит-тестов

## Как запустить

1. Открыть папку `lab02` в терминале.
2. Выполнить сборку:

```bash
dotnet build Lab02.sln
```

3. Запустить консольное приложение:

```bash
dotnet run --project src/Lab02/Lab02.csproj
```

4. Запустить тесты:

```bash
dotnet test Lab02.sln
```

Приложение выполнит замеры производительности всех коллекций и выведет подробный отчет с анализом результатов.
