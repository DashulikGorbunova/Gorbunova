# Лабораторная работа 3. Реализация коллекций и интерфейсов

**Тема**: реализация коллекций и интерфейсов

## Цель

Познакомиться с интерфейсами коллекций в C# (.NET), реализовать простые коллекции и понять их контракт и функциональность.

## Основные интерфейсы для изучения

- **IEnumerable** — поддержка перебора элементов (foreach)
- **ICollection** — базовые операции: добавление, удаление, подсчёт элементов
- **IList** — индексированный доступ, вставка, удаление по индексу
- **IDictionary<TKey, TValue>** — словарь, работа с ключами и значениями

## Задание

### 1. SimpleList

Реализован класс `SimpleList<T>`, который:
- Реализует интерфейсы `IEnumerable<T>`, `ICollection<T>`, `IList<T>`
- Внутренне использует динамический массив (без готовых коллекций List и т.п.)
- Поддерживает: добавление, получение/установку по индексу, удаление, поиск индекса
- Реализованы методы и свойства: `Count`, `Add`, `Remove`, `Clear`, `Contains`, `IndexOf`, `Insert`, `RemoveAt`

### 2. SimpleDictionary<TKey, TValue>

Реализован класс `SimpleDictionary<TKey, TValue>`, который:
- Реализует `IDictionary<TKey, TValue>` (собственный массив с цепочками)
- Поддерживает добавление, удаление и поиск по ключу
- Обеспечивает перебор (`IEnumerable<KeyValuePair<TKey, TValue>>`)
- Реализует поддержку `IReadOnlyCollection` / `IReadOnlyList` через свойства `Keys` и `Values`

### 3. DoublyLinkedList<T>

Реализована собственная версия двунаправленного связного списка с интерфейсом `IList<T>`:
- Поддерживает все операции `IList<T>`
- Оптимизирован для вставки и удаления в начале и конце списка
- Поддерживает двунаправленный обход

## Требования к тестам

- Написаны автоматизированные тесты, которые проверяют корректность всех операций
- Тесты детерминированные и воспроизводимые (одинаковые входные данные, одинаковые ожидаемые результаты)
- Использован фреймворк xUnit для тестирования

## Структура проекта

- **src/Lab03** – библиотека классов с реализациями:
  - `SimpleList.cs` – реализация списка на основе динамического массива
  - `SimpleDictionary.cs` – реализация словаря с хеш-таблицей
  - `DoublyLinkedList.cs` – реализация двунаправленного связного списка
- **tests/Lab03.Tests** – проект с юнит-тестами (xUnit):
  - `SimpleListTests.cs` – тесты для SimpleList
  - `SimpleDictionaryTests.cs` – тесты для SimpleDictionary
  - `DoublyLinkedListTests.cs` – тесты для DoublyLinkedList

## Используемые технологии и библиотеки

- **.NET 8.0**
- **xUnit** для юнит-тестов

## Как запустить

1. Открыть папку `lab03` в терминале.
2. Выполнить сборку:

```bash
dotnet build Lab03.sln
```

3. Запустить тесты:

```bash
dotnet test Lab03.sln
```

Все тесты должны пройти успешно, подтверждая корректность реализации всех коллекций.
