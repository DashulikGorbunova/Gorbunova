# Лабораторная работа 1. JSON, файлы и IDisposable (C#)

**Тема**: сериализация JSON, управление ресурсами и работа с файлами в C#.

### Цели работы

- **Использование атрибутов JSON** для управления процессом сериализации/десериализации.
- **Изучение паттерна IDisposable** для корректного освобождения ресурсов.
- **Работа с файловой системой** (чтение/запись файлов).
- **Сохранение и загрузка объектов** в/из файлов JSON.
- **Понимание разницы** между явным вызовом `Dispose()` и автоматическим освобождением через финализатор.

### Структура проекта

- **src/Lab01** – консольное приложение с основной логикой:
  - `Models/Person.cs` – класс с атрибутами JSON и простыми проверками.
  - `Services/PersonSerializer.cs` – сериализация/десериализация и работа с файлами.
  - `Services/FileResourceManager.cs` – пример работы с файлами и `IDisposable`.
  - `Services/Logger.cs` – простой логгер ошибок в файл `errors.log`.
  - `Program.cs` – маленькая демонстрация работы.
  - `Lab01.csproj` – файл проекта.
- **tests/Lab01.Tests** – проект с юнит-тестами (xUnit):
  - `PersonTests.cs` – тесты для класса Person.
  - `PersonSerializerTests.cs` – тесты для сериализации.
  - `FileResourceManagerTests.cs` – тесты для FileResourceManager.

### Используемые технологии и библиотеки

- **.NET 8.0**
- **System.Text.Json** для сериализации JSON (с атрибутами `[JsonPropertyName]`, `[JsonIgnore]`, `[JsonInclude]`).
- **xUnit** для юнит-тестов.

### Как запустить

1. Открыть папку `lab01` в терминале.
2. Выполнить сборку:

```bash
dotnet build Lab01.sln
```

3. Запустить консольное приложение:

```bash
dotnet run --project src/Lab01/Lab01.csproj
```

4. Запустить тесты:

```bash
dotnet test Lab01.sln
```

Приложение создаёт тестовый объект `Person`, сериализует его в JSON, выводит результат в консоль и демонстрирует базовую работу с файлами. Логирование ошибок ведётся в файл `errors.log` в кодировке UTF-8.
